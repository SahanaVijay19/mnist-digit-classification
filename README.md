# mnist-digit-classification
Built and optimized a CNN using PyTorch to classify high-dimensional handwritten digit data.

Explored the implementation of deep learning architectures for image recognition tasks to understand the mechanics of automated visual data processing.Tasked with building and optimizing a Convolutional Neural Network (CNN) using PyTorch to classify high-dimensional visual data. Architected a multi-layer CNN involving convolutional, pooling, and fully connected layers, applying Softmax activation and Cross-Entropy Loss for multi-class optimization. Achieved a test accuracy of over 99% by tuning hyperparameters, demonstrating a foundational ability to translate state-of-the-art theory into a functional neural network.
